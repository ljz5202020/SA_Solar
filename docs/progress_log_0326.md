### 2. SAM Edit 工作流验证

首次处理 "模型检测到但切割不准" 的情况：
- **单目标重切**: 标记 edit → SAM 在质心点重新分割 → 替换多边形
- **分裂重切**: 两个安装被合并成一个预测 → 用原始预测减去第一次 SAM 结果 → 在余量质心做第二次 SAM → 得到两个独立多边形
- **关键发现**: predictions_metric.gpkg 是 EPSG:32734，tiles 是 EPSG:4326，必须先转换 CRS 再计算像素坐标

### 3. SAM Mask 选择策略更新

- mask[1] (中间粒度) 仍是默认最佳选择

- 需要逐 case 确认，已在 G1685_6_1 验证此策略
