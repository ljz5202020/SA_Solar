# 太阳能板标注工作流迁移指南

## 从 Google Earth 网页版 → QGIS

> **项目背景**：南非开普敦屋顶太阳能板检测与时序标注（CV 训练数据集制作）

***

## 1. 环境准备

### 1.1 安装 QGIS

- 下载安装 QGIS（包含 OSGeo4W Shell，后续 GDAL 命令行会用到）


### 1.2 导入开普敦高清底图（WMS）

1. **图层 → 添加图层 → 添加 WMS/WMTS 图层 → 新建**
2. URL 填入：

```
https://cityimg.capetown.gov.za/erdas-iws/ogc/wms/GeoSpatial%20Datasets?
```

3. 连接后在图层列表选择对应年份图层（如 `Aerial Imagery 2024`、`Aerial Imagery 2025Jan`）
4. 图像编码格式选 **JPEG**（加载速度更快）
5. **时序对比**：同时加载多个年份图层，通过勾选显示/隐藏，快速对比同一屋顶在不同年份的状态

***

## 2. 导入现有 KML 标注文件

### 2.1 导入方式

- **最快**：直接把 `.kml` 文件拖入 QGIS 地图画布
- **规范**：图层 → 添加图层 → 添加矢量图层 → 选择文件


### 2.2 转换为可编辑格式

KML 在 QGIS 中不可直接编辑，必须先转换：
> 右键图层 → **导出 → 另存为要素** → 格式选 `GeoPackage` 或 `ESRI Shapefile`

### 2.3 处理偏移问题

导入后如发现多边形与底图有偏移（通常 2–5 米），原因是：

- **来源底图（如 Google Earth）存在绝对位置误差**，开普敦市政图精度更高
- **建筑物倾斜效应（Parallax）**：不同视角航拍导致屋顶像素位置偏移

**修复步骤：**

1. 开启编辑模式（✏️ 黄色铅笔）
2. `Ctrl + A` 全选所有多边形
3. 在**顶部水平工具栏**（非左侧面板）点击 🔀 **移动要素** 工具
4. 左键点击起点 → 拖动对齐 → 左键放下 → 保存

***

## 3. 网格任务管理

### 3.1 添加完成状态字段

1. 打开网格（Grid）图层属性表，开启编辑模式
2. 新建字段：名称 `is_done`，类型 **整型 (Integer)**
3. 用**字段计算器**把所有行批量赋初始值 `0`（避免 NULL 导致的 Invalid 状态）

### 3.2 设置复选框 Widget

> 图层属性 → **属性表单** → 选中 `is_done` 字段 → 小部件类型改为 **Checkbox** → 选中值=`1`，未选中值=`0`

### 3.3 隐藏已完成的网格（基于规则的符号化）

1. 图层属性 → **符号化** → 顶部改为 **基于规则 (Rule-based)**
2. 新建规则，Filter 填入（根据实际存储格式选一个）：

```sql
-- 如果字段存的是整数 0/1：
"is_done" IS NULL OR "is_done" = 0

-- 如果字段存的是文本 'false'/'true'：
"is_done" IS NULL OR "is_done" != 'true'

-- 万能反向写法（最稳定）：
"is_done" IS NULL OR "is_done" != 'true'
```

3. **调试技巧**：遇到 `Filter returned 0 feature(s)`，点击表达式编辑器里的 **All Unique** 按钮，查看字段里真实存储的值，再据此修改表达式

### 3.4 批量标记完成

1. 用框选工具选中一批格子
2. 打开属性表 → 切换到**表单视图** → 开启**多重编辑模式**
3. 对 `is_done` 字段打勾 → 保存
4. 已完成的格子立即从地图上隐身

***

## 4. 太阳能板标注

### 4.1 画多边形（标准流程）

1. 在左侧图层面板**单击选中**太阳能板图层
2. 点击 ✏️ **黄色铅笔**开启编辑模式
3. 点击顶部工具栏的 **添加多边形要素**（绿色多边形+加号图标）
4. 左键逐点点击，勾勒太阳能板轮廓
5. **右键**结束当前多边形，填写属性（或直接确定）
6. 完成一批后，点击铅笔图标 → 保存

### 4.2 图层顺序（看不见面板时检查）

- 太阳能板图层必须在图层列表**最顶部**，否则会被底图压住
- 符号化设置：填充颜色建议**半透明黄色（50%透明度）**，描边纯红色加粗


### 4.3 设置标注显示 ID

> 图层属性 → **标注 (Labels)** → 单一标注 → 值选 `Name` 字段 → 开启文字描边缓冲区（白色边框）提升可读性

***

## 5. 自动生成 Unique ID

格式要求：`G1190_001`（Grid ID + 三位序号）

### 5.1 空间连接（一次性操作）

处理工具箱搜索 **Join attributes by location**：

- 输入图层：太阳能板
- 连接图层：网格 Grid
- 谓词：`within`
- 要连接的字段：Grid 的 ID 字段（如 `grid_id`）


### 5.2 批量编号（每做完一个 Grid 执行一次）

1. 用 **Select by location** 选中当前 Grid 内所有面板
2. 打开**字段计算器**，勾选 **只更新选中要素**
3. 更新 `unique_id` 字段，表达式填入（替换 `G1190` 为实际 ID）：

```sql
concat( 'G1190', '_', lpad( to_string( @row_number + 1 ), 3, '0' ) )
```

## 6. 离线底图缓存（高性能标注）

### 6.1 用 GDAL 命令行下载指定区域

**准备 XML 配置文件** `capetown.xml`：

```xml
<GDAL_WMS>
  <Service name="WMS">
    <Version>1.1.1</Version>
    <ServerUrl>https://cityimg.capetown.gov.za/erdas-iws/ogc/wms/GeoSpatial%20Datasets?</ServerUrl>
    <Layers>Aerial_Imagery_2025Jan</Layers>
    <ImageFormat>image/jpeg</ImageFormat>
    <BBoxOrder>xyXY</BBoxOrder>
  </Service>
  <BlockSizeX>1024</BlockSizeX>
  <BlockSizeY>1024</BlockSizeY>
  <BandsCount>3</BandsCount>
  <Timeout>300</Timeout>
</GDAL_WMS>
```

**执行下载**（OSGeo4W Shell）：

```bash
gdal_translate \
  -projwin <xmin> <ymax> <xmax> <ymin> \
  -tr 0.000001 0.000001 \
  -co COMPRESS=JPEG -co TILED=YES \
  capetown.xml output_G1190.tif
```


### 6.2 多个 Tile 合并为 VRT 虚拟栅格

```bash
cd \tiles
gdalbuildvrt G1238_mosaic.vrt G1238_*.tif
# 生成多级概视图，提升缩放流畅度：
gdaladdo G1238_mosaic.vrt 2 4 8 16 32
```

将 `G1238_mosaic.vrt` 拖入 QGIS 即可，**不占额外磁盘空间**。

### 6.3 每个 Grid 的标注循环

```
下载当前 Grid 的 tiles
    ↓
gdalbuildvrt 合并 + gdaladdo 生成概视图
    ↓
拖入 QGIS，关闭在线 WMS 图层
    ↓
标注太阳能板多边形
    ↓
字段计算器批量生成 unique_id
    ↓
Grid 的 is_done 打勾 → 格子自动隐身
    ↓
QGIS 移除 VRT，删除 tiles 文件夹（释放磁盘）
    ↓
重复下一个 Grid
```

## 7. 常见问题速查

| 症状 | 原因 | 解决方法 |
| :-- | :-- | :-- |
| 画的多边形看不见 | 图层顺序/填充透明 | 把图层拖到最顶部；修改符号化填充颜色 |
| KML 导入有偏移 | 来源底图绝对精度差 | 全选后用移动要素工具整体平移对齐 |
| 过滤规则返回 0 个要素 | 字段数据类型不匹配 | 在表达式编辑器用 All Unique 查看真实值，加单引号或改用反向逻辑 |
| 高级数字化面板全灰 | 当前地图工具不对 | 先在顶部工具栏选中"移动要素"或"添加要素"工具 |
| 复选框显示 Invalid | 字段初始值是 NULL | 用字段计算器批量赋值 `0` |
| WMS 下载超时 | 请求区域太大 | 改用 GDAL 命令行分块下载 |
| VRT 加载卡顿 | 缺少概视图 | 运行 `gdaladdo` 生成多级缩略图缓存 |
